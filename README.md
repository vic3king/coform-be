## Description

Simple application for uploading .pdf and .txt files

Built with:
[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod

# seed if needed
$ npm run seed

# reset db if needed
$ npx prisma migration reset

# run after migration
$ npx prisma db push  
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
